# Selenium Project - Test Suite
This is a test suite for the Selenium project written in Java.

## Installation
- Install Java Development Kit (JDK)
- Install Maven build tool
- Install Selenium WebDriver
- Install a compatible web driver, such as ChromeDriver or GeckoDriver

## Setup
- Clone the Selenium project repository from GitHub
- Import the project into your preferred Java IDE
- Configure the Maven dependencies in the project's pom.xml file
- Set up the appropriate WebDriver for your chosen web driver
- Configure any other project-specific settings, such as base URL or test data

## Test Execution
- You can Run the test classes by clicking on the debug icon for all Suite or for only one Test case
- Review any failed tests and investigate and fix any issues

## Test Suite Contents
- **LoginTest**: Contains test cases related to user login functionality
- **RegistrationTest**: Contains test cases related to user registration functionality
- **SearchTest**: Contains test cases related to search functionality
- **FollowTest**: Contains test cases related to follow functionality
- **WishlistTest**: Contains test cases related to wishlist functionality
- **HoverTest**: Contains test cases related to Hover functionality
- **SliderTest**: Contains test cases related to Slider functionality
- **CurrencyTest**: Contains test cases related to order confirmation functionality
